### 1. 过古时间过滤 ✅
```python
def is_too_old(self, dt_string):
    """判断时间是否过于久远（超过20年前）"""
    parsed = datetime.strptime(dt_string, '%Y-%m-%d %H:%M:%S')
    now = datetime.now()
    return parsed < (now - timedelta(days=365*20))
```

### 2. "刚刚"时间过滤 ✅
```python
def is_just_now(self, dt_string):
    """判断时间是否是"刚刚"（与当前时间相差小于5分钟）"""
    parsed = datetime.strptime(dt_string, '%Y-%m-%d %H:%M:%S')
    now = datetime.now()
    diff = abs((now - parsed).total_seconds())
    return diff < 300  # 5分钟
```
